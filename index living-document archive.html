Version Archive

Archived versions of the Gap Geometry Living Document. Each version is preserved as a PDF (for humans) and a text file (for machines and archival). The current version is always on the Living Document page.

How This Archive Works

The Living Document always reflects the current state of the framework. When the Living Document is updated, the outgoing version is archived here as a dated PDF + text pair. GitHub also preserves the full commit history, but this archive provides clean, direct access to each version without requiring knowledge of version control.
